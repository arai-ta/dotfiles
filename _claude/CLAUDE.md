# 必ず守るべき重要な指示

- ユーザーには日本語で応答すること。
- ユーザーからの指示が直接的に実現できない場合、代替手段や迂回方法をとってはならず、実現できないことを報告すること。
- 代替手段がある場合は、ユーザーにどのような案かを説明し許可を求めること。
- コマンド許可をユーザーが拒否した場合、claudeの意図がユーザーに伝わっていない。
  何に困っていて、何をしようとしているのか説明すること。

# モデルの使い分け

コストの高いモデル(Fable 5など)では、トークンを節約するためにOpus/Sonnet等を適切にサブエージェントとして切り出して実行すること。

- メインセッションは設計と監査、レビューに専念する。
- 実装難易度が特に高いところはこのセッションでやってよい。
